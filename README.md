# padaflix
software de assinatura para delivery de produtos de padaria

## Configurar Frontend  <br>
- Pré-requisitos: NodeJs instalado E atualizado  <br>

Abra a pasta ze-auto no terminal e execute os seguintes comandos: <br>
```
cd frontend 
npm install
```

## Rodar frontend:  <br>
```
cd frontend 
npm run dev 
```
